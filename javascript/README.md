# todolist

> JavaScript • TodoMVC

这仅仅是一个模板，可以直接使用

## 技术点

- 1. 获取数据并显示到页面中 ok
- 2. 添加任务 ok
- 3. 删除任务（单个）ok
- 4. 更改任务状态 ok
- 5. 增加筛选
- 7. 计算未完成任务的数量
- 8. 清空已完成任务
- 9. 批量更改任务状态 ok
- 10. 修改任务名称
